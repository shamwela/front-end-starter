# Front-end Starter

## Main technologies

- Next.js
- TypeScript
- Tailwind CSS

## Benefits of using this

This starter will save you time and energy.

## How to use this

1. Click the green "Code" button on GitHub.
1. Click "Download ZIP".
1. Unzip the downloaded folder.
1. Rename the folder.
1. Open the folder in the terminal.
1. Run `npm i`.
